# torch.unsqueeze用法
```python
a = torch.rand((3,4,5))
b = a.unsqueeze(0)
print(b.shape)
c = a.unsqueeze(1)
print(c.shape)
d = a.unsqueeze(-1)
print(d.shape)
e = a.unsqueeze(3)
print(e.shape)
```
运行结果：  
torch.Size([1, 3, 4, 5])  
torch.Size([3, 1, 4, 5])  
torch.Size([3, 4, 5, 1])  
torch.Size([3, 4, 5, 1])  
可见，参数是从第一个数前算起并从0开始的，如果要插到最后面就用-1
